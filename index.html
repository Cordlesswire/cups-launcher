CUPS Launcher

A simple CUPS launcher for Ubuntu.

Just install it with:

sudo apt-add-repository ppa:paolorotolo/cups-launcher
sudo apt-get update
sudo apt-get install cups-launcher



A simple launcher.

It opens your local CUPS server in your favourite browser.





A Powerful Printing Center.

Manage your printers, jobs and classes with one click.





Compatible.

It works with all DE, like Unity and GNOME-Shell.





Designed for Unity.

CUPS Launcher supports Unity Shortcuts. Just a right-click on the icon and you'll able to go directly at the page you want.




Open Source.

Source code is available on Launchpad.

Developed by Paolo Rotolo.
